# 5 个小惊喜功能计划

## 项目背景

"课表 · 竹" 是一个温暖风格的课程表应用，已有 EasterEgg 随机鼓励语、WarmthBanner 每日问候、WeeklySummary 周总结、心情标签等惊喜功能。以下 5 个新惊喜将延续这种温暖、关怀的设计理念。

---

## 惊喜 1：学期倒计时胶囊 🎯

**触发方式**：在 TopBar 中显示一个小的倒计时胶囊标签

**功能描述**：
- 显示距离学期结束还有多少天
- 根据剩余天数显示不同的鼓励语：
  - `>60天`："还有 X 天，一步一步来"
  - `30-60天`："还有 X 天，坚持就是胜利"
  - `10-30天`："还有 X 天，胜利在望！"
  - `<10天`："最后 X 天，冲刺！"
- 点击可展开显示更详细的学期进度（已过周数/总周数）
- 使用已有的 `SemesterConfig` 数据计算，无需额外数据源

**实现要点**：
- 新建 `SemesterCountdown.tsx` 组件
- 在 TopBar 中集成
- 使用 Framer Motion 做展开/收起动画
- 颜色随倒计时阶段渐变（青色 → 橙色 → 红色）

---

## 惊喜 2：课程完结庆祝 🎉

**触发方式**：当某门课的最后一节课结束时（进度从 <100% 变为 100%）

**功能描述**：
- 检测到课程完结时，在页面中央播放一个轻量的五彩纸屑（confetti）动画
- 显示 "🎉 恭喜完成 [课程名]！" 的祝贺卡片
- 祝贺卡片 3 秒后自动消失
- 每门课只庆祝一次（localStorage 记录已庆祝的课程 ID）
- 动画使用纯 Canvas 或 CSS 实现，不引入额外库

**实现要点**：
- 新建 `CourseCompletionCelebration.tsx` 组件
- 在 WeekView/DayView 中检测当前课程进度变化
- 用 Canvas 实现简单的 confetti 粒子效果（50-80 个粒子）
- localStorage 存储已庆祝记录：`celebrated_courses: ["course_id_1", ...]`

---

## 惊喜 3：深夜关怀 🌙

**触发方式**：在 23:00 - 6:00 期间使用应用时

**功能描述**：
- 页面顶部出现一条柔和的月亮主题横幅："夜深了，早点休息吧 🌙"
- 横幅背景为深蓝渐变，带星星闪烁的 CSS 动画
- 提供"我知道了"按钮关闭，当晚不再显示
- 如果超过凌晨 1 点，文案变为："都凌晨了！明天还要上课呢 😴"
- 不影响正常使用，只是温柔的提醒

**实现要点**：
- 新建 `LateNightCare.tsx` 组件
- 在 layout.tsx 或首页中集成
- 使用 CSS `@keyframes` 实现星星闪烁
- localStorage 记录当晚是否已关闭：`late_night_dismissed_2026-04-30`

---

## 惊喜 4：课间小贴士 💡

**触发方式**：当前时间处于两节课之间的课间休息时段

**功能描述**：
- 在 WarmthBanner 区域下方，显示一条随机小贴士
- 贴士内容池（10+ 条）：
  - "站起来伸个懒腰吧 🙆"
  - "喝口水补充能量 💧"
  - "看看窗外休息一下眼睛 👀"
  - "深呼吸三次，放松一下 🌬️"
  - "下节课加油！💪"
  - "课间听首歌放松一下吧 🎵"
  - 等等
- 每个课间只显示一条，随机选择
- 带有淡入淡出动画，5 秒后自动消失

**实现要点**：
- 新建 `BreakTip.tsx` 组件
- 利用已有的课表时间计算逻辑判断是否在课间
- 在 WarmthBanner 下方集成
- 随机从贴士池中选取，避免连续重复（localStorage 记录上次显示的索引）

---

## 惊喜 5：节日彩蛋 🎊

**触发方式**：在特定节日/纪念日当天使用应用

**功能描述**：
- 在节日当天，TopBar 的 Logo 旁边显示对应的节日 emoji
- WarmthBanner 的问候语替换为节日祝福
- 预置的节日列表：
  - 🎆 元旦（1月1日）："新年快乐！新学期新开始"
  - 🧧 春节（农历正月初一，简化为固定日期或手动配置）："新年快乐，万事如意"
  - 🌸 情人节（2月14日）："今天也要好好上课哦"
  - 🎓 毕业季（6月）："毕业快乐！未来可期"
  - 🌕 中秋节："月圆人团圆，中秋快乐"
  - 🎃 万圣节："不给糖就捣蛋！"
  - 🎄 圣诞节："Merry Christmas！"
  - 🎂 用户生日（可在设置中配置）："生日快乐！今天你是主角 🎉"
- 节日 emoji 带有轻微的弹跳动画

**实现要点**：
- 新建 `FestivalEasterEgg.tsx` 组件
- 在 `lib/festivals.ts` 中定义节日数据（日期 + emoji + 祝福语）
- 在 TopBar 和 WarmthBanner 中集成
- 用户生日存储在 user-settings 中
- 农历节日简化处理：可使用固定公历近似日期或手动配置

---

## 实现顺序

1. 学期倒计时胶囊 — 独立组件，集成到 TopBar
2. 深夜关怀 — 独立组件，集成到 layout
3. 课间小贴士 — 依赖课表时间计算，集成到首页
4. 课程完结庆祝 — 依赖进度检测，集成到周/日视图
5. 节日彩蛋 — 需要节日数据 + 多处集成

## 技术原则

- 所有惊喜都使用现有技术栈（React + Framer Motion + Tailwind CSS）
- 不引入新的第三方库
- 遵循现有组件的代码风格和动画风格
- 使用 localStorage 做去重和状态记录
- 每个惊喜都是独立组件，可单独开关
