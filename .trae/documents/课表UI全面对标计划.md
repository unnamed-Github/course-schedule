# 课表应用 UI 全面对标计划（修订版）

> 对标参考：`https://gcngflqoco86.aiforce.cloud/app/app_4k18jbjfxg593`（妙搭版课表）
> 我们的应用：`https://course-schedule-one.vercel.app`（课表·竹）

---

## 一、对标分析总结（已纠正）

### 参考站（妙搭版）核心设计特征
1. **顶部导航**：5个文字链接 Tab（周视图/日视图/课程/作业/备忘），选中态蓝色加粗
2. **设置入口**：右下角浮动"设置"按钮，点击弹出设置面板（含"显示"/"数据"两个子Tab）
3. **周视图**：
   - 标题"周视图"（h2）
   - 左箭头 + 下拉选择器（第1-15周）+ 右箭头
   - "高亮开启/关闭"按钮
   - 问候横幅："最轻松的一天，好好过 🍃"（简洁一行文字）
   - 课程格子：纯色背景 + 白色文字，点击展开显示"暂无作业和备忘"
4. **日视图**：日期选择器（年/月/日 spinbutton），课程卡片横向布局
5. **课程视图**：
   - 标题"课程管理" + "共 9 门课程"
   - "添加课程"按钮在标题右侧
   - 课程卡片：顶部色条 + 课程名 + 老师·教室 + 进度条（X/15节）+ 编辑/删除图标按钮
6. **作业视图**：
   - 四宫格统计（全部/待提交/已提交/已过期）
   - "快速添加作业"按钮
   - 筛选标签（全部/待提交/已提交/已过期）
   - "添加作业"按钮
   - "默认设置"按钮
7. **备忘视图**：
   - 标题"课堂备忘"
   - "新增备忘"按钮
8. **暗色模式**：深蓝底色 + 柔和对比
9. **左上角**："由妙搭搭建"品牌标识

### 我们的应用当前状态（已纠正）
1. **顶部导航**：6个按钮式 Tab（周视图/日视图/课程/作业/备忘/设置），选中态蓝色填充白字
2. **设置入口**：作为顶部 Tab 之一，切换到设置视图（内嵌在 MainView 中）
3. **周视图**：
   - 无"周视图"标题
   - 左箭头 + 周次文字 + 右箭头（**无下拉选择器**）
   - **无高亮开关按钮**
   - WarmthBanner：绿色背景问候横幅 + 当前课程进度条
   - 课程格子：纯色背景 + 白色文字，点击展开显示老师/节次
4. **日视图**：箭头 + 日期文字导航，课程卡片有进度条
5. **课程视图**：
   - 标题"📚 我的课程 (X门)"
   - 导入/CSV/Excel 按钮在标题右侧
   - 课程卡片：顶部色条 + 课程名 + 老师·教室 + 进度小方格 + 心情标签 + 作业折叠 + 编辑/删除
6. **作业视图**：四宫格统计 + 筛选标签 + 作业列表（**无快速添加按钮**）
7. **备忘视图**：内联输入框 + 备忘列表 + 心情统计
8. **暗色模式**：已实现
9. **无品牌标识**
10. **⚠️ 没有底部导航栏**（BottomNav.tsx 文件存在但未在 layout.tsx 中使用）

## 二、具体改动计划

### 改动 1：周视图 - 添加周次下拉选择器 + 高亮开关按钮

**文件**：`src/components/WeekView.tsx`

**参考站设计**：
- 下拉选择器（combobox）选择周次（第1-15周）
- 左右箭头按钮快速切换
- "高亮开启/关闭"按钮

**我们的改动**：
1. 在周次切换区域添加 `<select>` 下拉选择器，显示"第X周"
2. 布局改为：左箭头 + 下拉选择器 + 右箭头
3. 在周次切换区域右侧添加高亮开关按钮（"高亮开启"/"高亮关闭"）
4. 高亮开关切换 `highlight_enabled` 设置

### 改动 2：问候横幅样式简化

**文件**：`src/components/WarmthBanner.tsx`

**参考站设计**：
- 简洁一行文字，如"最轻松的一天，好好过 🍃"
- 浅色背景，无复杂装饰

**我们的改动**：
1. 去掉绿色背景（`#F0FDF4`），改为 `var(--bg-card)` + 左侧暖色色条
2. 问候语保持现有逻辑，但去掉鼓励语副标题
3. 保持当前课程进度条功能（这是我们的优势功能）

### 改动 3：日视图 - 添加日期选择器

**文件**：`src/components/DayView.tsx`

**参考站设计**：
- 日期选择器使用 spinbutton（年/月/日）
- 点击可弹出日期选择

**我们的改动**：
1. 在日期文字上添加点击弹出 `<input type="date">` 的功能
2. 保持现有箭头 + 日期文字布局

### 改动 4：课程视图 - 标题和布局调整

**文件**：`src/app/courses/page.tsx`

**参考站设计**：
- 标题："课程管理" + "共 9 门课程"
- "添加课程"按钮在标题右侧
- 课程卡片：顶部色条 + 课程名 + 老师·教室 + 进度条（X/Y节）+ 编辑/删除图标按钮

**我们的改动**：
1. 标题改为"课程管理" + "共 X 门课程"
2. "添加课程"按钮移到标题右侧（目前导入/导出按钮在那里，需要重新排列）
3. 进度显示添加进度条（X/Y 节），保留进度小方格作为可选展开
4. 保持心情标签和作业折叠区（我们的优势功能）

### 改动 5：作业视图 - 添加快速添加作业按钮

**文件**：`src/components/AssignmentsView.tsx`

**参考站设计**：
- "快速添加作业"按钮
- "添加作业"按钮
- "默认设置"按钮

**我们的改动**：
1. 在统计卡片区域上方添加"快速添加作业"按钮
2. 添加作业弹窗功能（使用现有 Modal 组件）
3. 保持现有四宫格统计和筛选标签

### 改动 6：备忘视图 - 标题和按钮调整

**文件**：`src/components/MemosView.tsx`

**参考站设计**：
- 标题："课堂备忘"
- "新增备忘"按钮

**我们的改动**：
1. 标题改为"课堂备忘"
2. 在标题右侧添加"新增备忘"按钮，点击弹出备忘编辑弹窗
3. 保持现有备忘列表和心情统计

### 改动 7：设置入口改为浮动按钮 + 弹窗

**文件**：`src/app/settings/page.tsx`、新增 `src/components/SettingsModal.tsx`、`src/components/TopBar.tsx`、`src/components/ViewContext.tsx`、`src/app/layout.tsx`

**参考站设计**：
- 右下角浮动"设置"按钮
- 点击弹出设置面板（含"显示"/"数据"两个子Tab）
- 设置面板内有：暗色模式开关、高亮开关、导入导出等

**我们的改动**：
1. 从 TopBar 的 TABS 中移除"设置"Tab
2. 从 ViewContext 的 ViewType 中移除 'settings'
3. 从 MainView 中移除 settings 视图渲染
4. 创建 `SettingsModal.tsx` 组件，将设置页内容迁移到弹窗，分为"显示"和"数据"两个子Tab
5. 在 layout.tsx 中添加右下角浮动设置按钮
6. 保留 `/settings` 路由页面（兼容直接访问）

### 改动 8：全局样式微调

**文件**：`src/app/globals.css`、`src/components/TopBar.tsx`

**参考站设计**：
- Tab 导航使用链接样式（无背景色，选中态蓝色加粗）
- 整体更紧凑

**我们的改动**：
1. TopBar 的 Tab 样式从按钮式（蓝色填充白字）改为链接式（选中态蓝色加粗下划线）
2. 确保暗色模式与参考站一致
