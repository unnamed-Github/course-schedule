## 妇女节改写的设计思路

三八妇女节全称"国际劳动妇女节"，纪念的是 1908 年纽约纺织女工罢工。原始精神是致敬劳动女性的坚韧与尊严，而非后来附加的各种主义。

**坚决删除：**
- 「女人不是天生的，而是后天成为的」— 波伏娃（存在主义女权理论核心句，用户排斥）
- 「她力量，不可限量」— 佚名（营销 slogan 味）

**替换为劳动女性视角的朴素致敬：**
- "每一位认真生活的女性，都值得被看见 — 佚名"
- "妈妈的手，既能缝补衣裳，也能撑起半边天 — 佚名"
- "从纺织女工到今天，一条走了很久的路 — 佚名"

风格定位：温暖、朴素、不喊口号、不贴标签。

### 二、刷新机制改造

将 `getDailyQuote` 中的选取从 date-seeded 改为 `Math.random()` 加权随机：

```ts
// 旧：date-seeded，同日同句
const seed = date.getFullYear() * 10000 + month * 100 + day
let pick = seed % totalWeight
for (const c of candidates) { pick -= c.weight; if (pick < 0) return c.quote }

// 新：Math.random() 加权随机，每次刷新随机
let pick = Math.random() * totalWeight
for (const c of candidates) { pick -= c.weight; if (pick < 0) return c.quote }
```

WarmthBanner 无需改动，`useMemo([schedules])` 保证同页面内不跳动。
