# 作业界面 UI 改进计划

## 概述
将现有作业界面改造为符合新设计稿的样式，包含快速添加区域、改进的统计卡片和作业列表展示。

## 目标文件
- `/Users/jack/Documents/trae_projects/Timer/src/components/AssignmentsView.tsx`

## 主要改进点

### 1. 新增可折叠的「快速添加作业」区域
- 顶部区域，默认展开
- 包含输入框：作业标题、选择课程、截止日期
- 「更多选项」展开按钮（可隐藏/显示描述、提醒设置）
- 「按 Enter 键快速添加」提示
- 右侧「添加」按钮

### 2. 统计卡片保持
- 保留 4 列布局：全部、待提交、已提交、已过期

### 3. 调整过滤器按钮位置
- 过滤器按钮移至快速添加区域下方
- 右侧新增设置图标和「+ 添加作业」按钮

### 4. 作业卡片重新设计
- 更简洁的布局
- 左侧课程颜色圆点 + emoji（📝）
- 课程名称、截止时间
- 右下角显示倒计时（如「提前 1 小时」）
- 完成状态切换按钮在右侧

### 5. 功能增强
- 新增倒计时计算函数（显示距截止时间还有多久）
- 快速添加支持 Enter 键提交
- 更好的视觉层次和间距

## 修改内容

### AssignmentsView.tsx
1. 新增 `quickAddExpanded` 状态（默认 true）
2. 重构添加作业逻辑，支持快速添加区域
3. 新增 `formatCountdown` 函数计算倒计时显示
4. 调整组件布局顺序
5. 更新作业卡片样式
6. 保持原有功能完整性（编辑、删除、状态切换等）

## 技术要点
- 遵循现有代码风格和样式约定
- 使用现有 CSS 变量
- 保持 TypeScript 类型安全
- 无新依赖引入
