# 周视图课程展开详情 - 添加作业和备忘显示

## 问题描述

周视图点击课程方格展开后，只显示教师、节次和操作按钮（取消本课/恢复本课），**看不到该课程的作业和备忘**。

数据层面没有问题——`assignments` 和 `memos` 已经在 `loadData()` 中加载到了 state，只是展开详情区域没有渲染它们。

参考妙搭版课表的设计：点击课程格子展开后应显示"暂无作业和备忘"（即使为空也要有这个区域提示）。

## 现状分析

### WeekView 展开详情当前内容（第 352-413 行）
1. 教师信息（`<User>` 图标 + 教师名）
2. 节次信息（`<Clock>` 图标 + 第X-Y节）
3. 操作按钮区（取消本课 / 恢复本课 / 撤销下课）

### 已有但未使用的数据
- `assignments: Assignment[]` — 已在 state 中，含 `course_id`、`title`、`due_date`、`status`
- `memos: Memo[]` — 已在 state 中，含 `course_id`、`content`、`mood_emoji`、`mood_tags`
- `getCourseAssignmentCount(courseId)` — 已有函数，统计待完成作业数

### 课程详情页参考（`/courses/[id]/page.tsx`）
作业显示：标题 + 倒计时 + 截止日期 + 完成状态切换
备忘显示：emoji + 内容 + 心情标签 + 时间

## 实现方案

### 修改文件
- `src/components/WeekView.tsx`

### 具体改动

#### 1. 在展开详情中添加作业区域

在"课程信息"（教师/节次）和"操作按钮"之间，新增作业列表区域：

- 筛选当前课程的作业：`assignments.filter(a => a.course_id === course.id)`
- 待提交作业优先显示，已提交的灰色显示
- 每条作业显示：标题 + 截止倒计时（如"3天后截止"/"已逾期"）
- 作业状态图标：待提交用空心方块，已提交用绿色勾
- 如果没有作业，显示"暂无作业"

#### 2. 在展开详情中添加备忘区域

紧跟在作业区域之后：

- 筛选当前课程的备忘：`memos.filter(m => m.course_id === course.id)`
- 每条备忘显示：emoji + 内容摘要（truncate）+ 心情标签
- 最多显示 3 条备忘，避免展开区域过长
- 如果没有备忘，显示"暂无备忘"

#### 3. 空状态统一提示

如果作业和备忘都为空，显示一行简洁提示："暂无作业和备忘"（与妙搭版一致）

#### 4. 新增辅助函数

复用课程详情页的 `countdown()` 逻辑，在 WeekView 中添加本地版本：
```ts
function countdown(dueDate: string): string {
  const diff = new Date(dueDate).getTime() - Date.now()
  if (diff < 0) return '已逾期'
  const days = Math.floor(diff / 86400000)
  if (days > 0) return `${days}天后`
  const hours = Math.floor(diff / 3600000)
  if (hours > 0) return `${hours}小时后`
  return '即将截止'
}
```

#### 5. 需要新增的 import

在 WeekView 顶部添加图标 import：
- `ClipboardList`（作业图标）
- `Pin`（备忘图标）
- `Check`、`Square`（作业完成状态图标）

├─────────────────────────────┤
│ 📋 作业 (2)                  │  ← 新增：作业标题
│   ☐ 作业1 · 3天后截止        │  ← 新增：待提交作业
│   ☐ 作业2 · 已逾期           │  ← 新增：逾期作业

│ 📌 备忘 (1)                  │  ← 新增：备忘标题

## 注意事项

1. 展开区域在课程方格内部，背景色是课程颜色，文字为白色，需保持白色文字风格
2. 作业和备忘的分割线使用 `border-white/20`（与现有分割线一致）
3. 备忘最多显示 3 条，避免展开区域过长影响网格布局
4. 已提交的作业用 `opacity-50` 降低视觉权重
5. 逾期作业用醒目颜色提示（在白色文字环境下用 `#FCA5A5` 浅红色）
